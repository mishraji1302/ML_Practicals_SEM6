💻 Machine Learning Practicals - Iris Dataset

This repository contains practical experiments based on the Iris Dataset for understanding fundamental concepts of Data Preprocessing, Visualization, Statistical Analysis, and Feature Scaling** in Machine Learning.

 📌 Experiments Included

 🔹 Experiment 01: Data Preprocessing and Visualization

* Data cleaning and normalization
* Understanding Iris dataset
* Data visualization using:

  * Scatter plots
  * Histograms
  * Box plots
* Identifying outliers and data distribution

---

🔹 Experiment 02: Statistical Analysis and Feature Scaling

* Statistical analysis:

  * Mean
  * Median
  * Standard Deviation
* Feature scaling techniques:

  * Min-Max Normalization
  * Z-score Standardization
* Training ML model (KNN)
* Performance comparison before and after scaling

---

📊 Dataset Used

* **Iris Dataset**
* 150 samples
* 3 classes:

  * Setosa
  * Versicolor
  * Virginica
* 4 features:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
